# xplat [hidden]

Add xplat for NativeScript Angular.

## Usage

```bash
nx generate xplat ...

```

## Options

### --npmScope

Alias(es): wn

Type: `string`

The npm scope to use.

### --prefix

Alias(es): p

Type: `string`

The prefix to apply to generated selectors.

### --skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
