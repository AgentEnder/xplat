# ng-add [hidden]

Add xplat ionic to an existing Nx Workspace.

## Usage

```bash
nx generate ng-add ...

```

## Options

### --platforms

Type: `string`

Target platforms to support.

### --prefix

Alias(es): p

Type: `string`

The prefix to apply to xplat generated selectors.

### --skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
