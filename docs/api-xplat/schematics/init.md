# init

Init xplat architecture.

## Usage

```bash
nx generate init ...

```

## Options

### --framework

Type: `string`

Frontend framework to support.

### --npmScope

Alias(es): wn

Type: `string`

The npm scope to use.

### --platforms

Type: `string`

Target platforms to support.

### --prefix

Type: `string`

The prefix to apply to generated selectors.

### --skipFormat

Default: `false`

Type: `boolean`

Skip formatting files

### --skipInstall

Default: `false`

Type: `boolean`

Skip installing dependency packages.
