# ng-add [hidden]

Add xplat to an existing angular configured Nx Workspace

## Usage

```bash
nx generate ng-add ...

```

## Options

### --framework

Type: `string`

Target frameworks to support.

### --platforms

Type: `string`

Target platforms to support.

### --prefix

Type: `string`

The prefix to apply to xplat generated selectors.

### --skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
