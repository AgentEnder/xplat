# service

Create a service configured for xplat projects and shared code.

## Usage

```bash
nx generate service ...

```

## Options

### --feature

Type: `string`

Target feature

### --name

Type: `string`

Service name

### --platforms

Type: `string`

Target platforms

### --projects

Type: `string`

Target projects

### --skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
