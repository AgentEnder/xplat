# pipe

Create a pipe configured for xplat projects and shared code.

## Usage

```bash
ng generate pipe ...

```

## Options

### feature

Type: `string`

Target feature

### name

Type: `string`

Pipe name

### platforms

Type: `string`

Target platforms

### projects

Type: `string`

Target projects

### skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
