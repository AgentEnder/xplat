# ng-add [hidden]

Add xplat angular to an existing Nx workspace.

## Usage

```bash
ng generate ng-add ...

```

## Options

### platforms

Type: `string`

Target platforms to support.

### prefix

Alias(es): p

Type: `string`

The prefix to apply to xplat generated selectors.

### skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
