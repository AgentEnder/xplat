# directive

Create a directive configured for xplat projects and shared code.

## Usage

```bash
ng generate directive ...

```

## Options

### feature

Type: `string`

Target feature

### name

Type: `string`

Directive name

### platforms

Type: `string`

Target platforms

### projects

Type: `string`

Target projects

### skipFormat

Default: `false`

Type: `boolean`

Skip formatting files
